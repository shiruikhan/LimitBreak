# Visão Geral do LimitBreak

LimitBreak é um aplicativo web pensado para desenvolvedores que desejam acompanhar e aprender treinos de academia. Ele combina aprendizado de exercícios com organização de treinos pessoais.

## Público-alvo

- Desenvolvedores que buscam uma rotina de treino.
- Usuários que preferem um guia de treino direto no navegador.
- Times de dev que querem um projeto de autoaperfeiçoamento físico.

## Componentes principais

- Catálogo de exercícios.
- Visualização de detalhes do exercício.
- Perfil de treino individual.
- Calendário de atividades.

## Motivação

O app será usado principalmente para fins pessoais da equipe, servindo como referência de treinos e como um pequeno projeto para aprendizado de desenvolvimento web e integração com APIs.
